# Coalition Simulator — Game of Thrones

Parliamentary government coalition simulator based in the Game of Thrones universe. Build a governing majority, assign Small Council portfolios, and test to see whether your cabinet passes investiture.

## How to play
1. Pick Houses until you hit a legislative majority (101 of 200 seats).
2. Assign every cabinet portfolio position to a coalition member.
3. Click **Evaluate** to freeze the attempt and see investiture status, government type, policy connectedness, and portfolio proportionality.
4. Click **Reset** to start a new attempt.

## Files
- `index.html` — layout and markup.
- `styles.css` — styling and layout helpers.
- `data.js` — houses, seats, and policy positions, portfolio list, seat details.
- `app.js` — game logic, rendering, evaluation logic.
